# Valient RNG

Valient RNG is a browser-based RNG game where players roll mysterious Vessels, craft powerful Artifacts, and chase ultra-rare pulls.

## Features

- Roll for Vessels
- Auto-Roll System
- Multiple Rarity Tiers
- Vessel Crafting
- Artifact Progression
- Luck Upgrades
- Rare Event System
- Cosmic-Themed Progression
